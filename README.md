# ITU-ACM-20-21-Spring-C-Programlama

## Gereklilikler
1. Ubuntu Terminal & Text Editor
2. Zoom hesabı

## Amaç
Katılımcılara basit C programlama bilgisi kazandırmak ve katılımcıları karmaşık programlama konularına hazırlamak.
